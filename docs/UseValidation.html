Class: UseValidation

UseValidation(value, validations) → {Object}

new UseValidation(value, validations) → {Object}

Хук для валидации принимаемых данных из input.
Parameters:
Name Type Description
value string Значение из инпута, которое вводит пользователь.
validations "isEmpty: true" Объект с ключом для валидации, проверяет пустое ли поле.
Source:
  • hooks/useValidation.js, line 15
Returns:
Возвращает объект данных с состояниями - isEmpty пустое ли поле, inputValid общее состояние валидации инпута, isEmptyError текстовая ошибка о пустом поле. {isEmpty: true/false, inputValid: true/false, isEmptyError: ""/"Поле не может быть пустым"}.
Type
Object
Example
Можно указывать любые ключи, и валидировать данные так, как от этого требует задача.
{isEmpty: true, minLength: 2, maxLength: 15, nameValid: true} - Проверка на пустоту поля, минимальную и максимальную длину полученных данных, а также проверка корректности ввода имени
